## Ejecutar localmente

Clone el proyecto

```bash
  git clone https://github.com/itskreisler/itskreisler.github.io.git
```

Ir al directorio del proyecto

```bash
  cd itskreisler.github.io
```

Instalar dependencias

```bash
  npm install
```

Iniciar el servidor

```bash
  npm run dev
```
